Gitについて

Gitとは？

Gitとはファイルのバージョンの管理をするツールのことでGitを使うことで古いバージョンに戻せたり、ファイルの管理、編集が簡単に出来るのです。なので不具合などが起こったときに効率的に作業ができるツールです。

Gitって何が出来るの？

  • バージョンの変更が容易
  • 新旧ファイルの一括管理
  • 編集履歴の共有
  • みんなで編集したものの統合ができる

Git hubとは？

Git hubではエンジニアの人たちが公開したいプログラムなどを共有できるソフトで、公開した人以外も修正を加えることができる。

htmlについて

htmlとは？

htmlとは略称で、Hyper Text Markup Languageというのが正式名称です。

index.htmlとは

ホームページなどを作る際にそのページの内容に応じて様々な名称のhtmlファイルを作成します。「例えば会社であればcampany.htmlなど」
しかし,必ず作らなければならないといけないファイルがあるのです。それがindex.htmlファイルです。
